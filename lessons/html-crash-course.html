دورة HTML/CSS مكثفة - أنشئ موقعك الأول

🎯 ما ستتعلمه في هذا الدرس:

  • فهم أساسيات HTML (العناصر، السمات، الهيكل)
  • استخدام CSS للتنسيق (الألوان، الخطوط، الهوامش)
  • بناء تخطيط باستخدام Flexbox
  • جعل الموقع متجاوباً مع الجوال

✍️ مثال عملي سريع:

<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>موقعي الأول</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f7ff;
            margin: 0;
            padding: 0;
        }
        header {
            background: #0d3b66;
            color: white;
            padding: 20px;
            text-align: center;
        }
        .container {
            width: 80%;
            margin: 20px auto;
            padding: 20px;
            background: white;
            border-radius: 10px;
            box-shadow: 0 0 10px rgba(0,0,0,0.1);
        }
    </style>
</head>
<body>
    <header>
        <h1>مرحباً بالعالم!</h1>
    </header>
    <div class="container">
        <p>هذا أول موقع لي على الإنترنت.</p>
    </div>
</body>
</html>

📂 المشروع العملي:

سنبني معاً موقع بطاقة شخصية تفاعلية يشمل:

  1. صورة شخصية
  2. معلومات التواصل
  3. قائمة بالمهارات
  4. تصميم أنيق متجاوب
